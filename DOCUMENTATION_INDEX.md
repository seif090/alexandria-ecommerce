## 🎬 Running a Demo - Exact Steps

### **Before Demo (30 min before)**
1. ☐ Open **DEMO_REFERENCE_CARD.md** (printed or on screen)
2. ☐ Follow "DEMO STARTUP" section
3. ☐ Verify "http://localhost:4200" loading
4. ☐ Have all credentials handwritten in visible location
5. ☐ Pre-seed demo data (optional, saves 2 min during demo)

### **During Demo (20 minutes)**
1. ☐ Follow **CLIENT_DEMO_CHECKLIST.md** phases exactly
2. ☐ Use **DEMO_REFERENCE_CARD.md** for each phase's script
3. ☐ Hit any snags? Check DEMO_REFERENCE_CARD troubleshooting
4. ☐ Reference **IMPLEMENTATION_SUMMARY.md** for talking points

### **After Demo**
1. ☐ Run **ADMIN_DASHBOARD_SETUP.md** query examples for follow-up
2. ☐ Send client **README.md** for further reading
3. ☐ Offer **TECHNICAL_ARCHITECTURE.md** to their engineers
